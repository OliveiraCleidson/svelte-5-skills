## [0.0.1] - 2026-04-02

### Added

- **svelte5-init** — Skill for scaffolding production-ready Svelte 5 + SvelteKit projects with 6 deployment targets (Cloudflare, Vercel, Netlify, Node, Static, SPA), Tailwind CSS v4, and TypeScript strict mode.
- **svelte5-review** — Skill for expert code review of Svelte 5 components covering rune correctness, template syntax, state management, cohesion, TypeScript safety, and performance.
- **sveltekit-review** — Skill for architectural review of SvelteKit applications covering routing, load functions, form actions, rendering modes, hooks, security, performance, and accessibility.
- MIT License.
- README with installation instructions via `npx skills add`.
